# 独立版虚拟猫咪伴侣

这是一个完全独立的虚拟猫咪伴侣应用程序，无需安装任何依赖即可运行。

## 特性

- 可爱的白色猫咪在屏幕上随机移动
- 当鼠标靠近时，猫咪会跟随鼠标光标
- 当鼠标移动缓慢时，猫咪会坐在鼠标光标上
- 当快速移动鼠标时，猫咪会返回随机移动
- 带有摆动尾巴动画效果的猫咪
- 透明、可穿透的窗口，始终置顶显示
- 无可见的命令行界面
- 显示您输入的字符在猫咪肚子上
- 不活动时显示默认符号 (?, !, 或 :)

## 如何运行

只需双击 `YourPet.exe` 文件即可运行应用程序。

## 便携性

这个exe文件是完全自包含的，可以复制到任何位置（如USB驱动器、其他计算机）并独立运行。

## 控制方式

- 在屏幕上移动鼠标与猫咪互动
- 在键盘上输入字符，会在猫咪肚子上显示
- 快速移动鼠标使猫咪离开
- 通过任务管理器结束进程来关闭应用程序
